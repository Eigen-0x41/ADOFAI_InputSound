# "A Dance of Fire And Ice" Input Sound MOD
ADOFAIでキー入力時にヒット音を鳴らすMOD。

**現在は非同期入力を使用していないと機能しません。**

キー入力時にヒット音を鳴らす関係上、</br>通常のヒット音は無音化させています。

# 使用方法
+ UnityModManagerをインストールする。
+ [Releases](https://github.com/Eigen-0x41/ADOFAI_InputSound/releases)より**InputSound.zip**をダウンロード。
+ UnityModManagerを開き、**Mods**タグ内にある**Drop zip files here**にダウンロードした**InputSound.zip**をドラッグアンドドロップする。
+ ゲームを起動する。

## 無効化したい場合
ゲーム内のUnityModManagerより</br>**On/Off**の欄で無効化してからゲームを再起動して下さい。</br>なお、v1.1.1よりゲームの再起動は必要ありません。
