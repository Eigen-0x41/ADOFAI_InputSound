# "A Dance of Fire And Ice" Input Sound MOD
ADOFAIでキー入力時にヒット音を鳴らすMOD。

**現在は非同期入力を使用していないと機能しません。**

キー入力時にヒット音を鳴らす関係上、</br>通常のヒット音は無音化させています。

# 使用方法
1. UnityModManagerをインストールする。
2. [Releases](https://github.com/Eigen-0x41/ADOFAI_InputSound/releases/latest)より**InputSound.zip**をダウンロード。
3. UnityModManagerを開き、**Mods**タグ内にある**Drop zip files here**にダウンロードした**InputSound.zip**をドラッグアンドドロップする。
4. ゲームを起動する。

# 更新方法
UnityModManagerより"Check Updates"を押した後、</br>
"Input Sound"を右クリックすると"Update To (バージョン番号)"という項目があれば、その項目を押すことで更新することができます。

バージョン1.2.0以前のバージョンでは</br>[使用方法](#使用方法)の2番目以降の手順で更新することができます。</br>

## 無効化したい場合
ゲーム内のUnityModManagerより</br>**On/Off**の欄で無効化してからゲームを再起動して下さい。</br>なお、v1.1.1よりゲームの再起動は必要ありません。

## その他
- ホールドヒットサウンドのリリース時ノーツ音はオフセット値が大きめに設定されているため実装しません。
